# ATM Kiosk Project (Python)

## Overview

This project is a simple **ATM Kiosk simulation** built using Python.
It allows a user to enter a PIN and perform basic ATM transactions such as checking balance, withdrawing money, and depositing money.

The program runs in the **command line (console)** and demonstrates basic Python concepts like:

* Conditional statements
* Loops
* User input
* Variables
